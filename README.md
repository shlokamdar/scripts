
# Bash Scripts Collection

A curated collection of Bash shell scripts for learning, automation, and real-world DevOps workflows.
This repository serves as both a practice space and a reusable script library for Linux, cloud, and deployment tasks.

---

## Repository Overview

This repository contains scripts covering core Bash concepts, system administration tasks, web server deployment, and automation patterns commonly used in DevOps and cloud environments.

The scripts are written to be:

* Easy to read
* Easy to modify
* Practical and reusable

---

## Contents

### Bash Fundamentals

* `first-few-commands.sh` – Basic Linux and Bash commands
* `conditional.sh` – Conditional statements in Bash
* `while-loop.sh` – While loop examples
* `nested-loop.sh` – Nested loop logic
* `50files.sh` – Create multiple files using a script

### Web Server Installation & Deployment

* `install-nginx.sh` – Install and configure Nginx
* `apache-install.sh` – Install Apache web server
* `deploy-on-nginx.sh` – Application deployment on Nginx
* `deploying-on-apache.sh` – Application deployment on Apache

### Automation & Validation

* `preflight_check.sh` – System preflight checks before deployment
* `validate_deployment.sh` – Validate deployment status
* `backup_monitor.sh` – Monitor backup processes

### Networking & File Transfer

* `scp-to-ec2.sh` – Securely copy files to an EC2 instance
* `multiple-servers-scp.sh` – Copy files to multiple servers using SCP
* `http-code.sh` – Check HTTP response codes

## Getting Started

### Clone the Repository

```bash
git clone https://github.com/<your-username>/<repo-name>.git
```

### Make Scripts Executable

```bash
chmod +x script-name.sh
```

### Run a Script

```bash
./script-name.sh
```

Some scripts may require:

* sudo privileges
* A Linux-based operating system (Ubuntu recommended)
* Internet connectivity
